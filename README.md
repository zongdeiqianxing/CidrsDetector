# CidrsDetector
根据ip或cidr批量扫描网段的协程扫描器。

可直接使用oneforall截取的ip列作为扫描目标，程序将根据所有ip生成cidr网段并扫描开放端口。
其中需要注意-d参数，如指定-d参数，默认扩展到整个c段；如不指定c段，脚本取文件内该网段出现的范围并左右扩展10个ip位。举例：文件内有1.1.1.20，1.1.1.50，那么程序将生成1.1.1.10-1.1.1.60的ip范围。
